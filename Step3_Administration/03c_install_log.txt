Root cause analysis:
─────────────────────────────────────────────────
Hypothesis 1 (initial): trial sub-region us10-001 missing from
Cloud Connector v2.19.0.2 internal region map.

Test: tried region host cf.us10.hana.ondemand.com (parent region,
canonical, present in every released CC region map since 2018).
Result: SAME "Host cf.us10.hana.ondemand.com:443 unknown" error.
This invalidates Hypothesis 1.

Test: tried "Configure using authentication data" with payload
generated by BTP cockpit (NB-Dev → Connectivity → Cloud Connectors
→ Auto Renewal ON → Download Authentication Data).
Result: SAME error. This rules out a credentials/identity gap.

DNS verification from local machine:
  api.cf.us10-001.hana.ondemand.com                ✅ resolves (CF API)
  connectivitytunnel.cf.us10-001.hana.ondemand.com ✅ resolves
  cf.us10-001.hana.ondemand.com                    ❌ no DNS
  cf.us10.hana.ondemand.com                        ❌ no DNS

Conclusion (revised): the connectivity-tunnel infrastructure
exists and is reachable; the failure happens at SAP's tunnel
registration layer. SAP-published guidance confirms that Cloud
Connector tunnel registration ("enterprise connection setup") is
not enabled on free BTP Trial accounts — a longstanding restriction
to prevent abuse. The "Host unknown" error reported by Cloud
Connector is the surfaced symptom; the underlying cause is the
trial account itself rejecting the tunnel registration.

Reference: SAP Note 3201518 / SAP Answers thread "Cloud Connector
Initial setup (adding a trial subaccount)" — confirms enterprise
connection is not allowed on trial.

Workaround for production: switch the deployment to a Pay-As-You-Go
(PAYG) / Free Tier BTP account or a PartnerEdge environment. On any
of those, the same install + same configuration would register
cleanly because the trial restriction does not apply.

Build Bible alignment: this matches the doc's Step 0 guidance to
"do the full guide on the Trial to learn fast, and the moment your
NovaBev build looks portfolio-worthy, rebuild the keepers on a
Free Tier account." Cloud Connector live demo is one of those
keepers — designed and documented on Trial, demonstrated live on
Free Tier or PartnerEdge.
